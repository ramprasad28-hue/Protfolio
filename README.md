## ✨ Features

- **Hero Section** — Animated availability badge, serif heading, skill tags
- **About Section** — Bio, stats (Year, Projects, Domains), social links
- **Skills Section** — Tabbed skill categories with animated progress bars
- **Projects Section** — Featured project cards with tech stack chips
- **Contact Section** — Validated contact form + direct contact cards
- **Responsive** — Works on Desktop, Tablet, and Mobile
- **No Framework** — Pure Vanilla HTML, CSS, and JavaScript only
