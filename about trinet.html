01 — Components

Four Systems, One Shield.

TRINETRA integrates four independent components that work in concert to detect, trap, and block attackers.

🔴
Red Team

Attacker Simulation

Simulates real-world attack behavior using SYN, FIN, NULL, and Xmas scans. Probes the network to find vulnerabilities — just like an actual threat actor would.

🔵
Blue Team

Defender Monitoring

Continuously monitors traffic using Python and the Scapy library. Analyzes TCP/IP flags, detects abnormal packet behavior, and logs every suspicious event.

🍯
Honeypot

Deception & Intelligence

Masquerades as a real vulnerable machine to lure attackers. Captures their IP addresses, tools, and behavior patterns — feeding that intelligence back to the Blue Team.

📡
IoT Device

Real-Time Hardware Alerts

ESP32/ESP8266 triggers a buzzer, LED, or message notification the instant a threat is detected — bringing physical, real-time awareness to cyber events.

02 — System Flow

How It Works.

From attack simulation to automatic blocking — the full pipeline.

1
Red Team
Attack Simulation Begins

The Red Team uses network scanning techniques — SYN, FIN, NULL, Xmas scans — to probe the target system and find open ports or vulnerabilities, exactly as a real attacker would.

2
Honeypot
Attacker Gets Trapped

The Honeypot presents itself as a real vulnerable machine. Attackers interact with it, unknowingly revealing their tools, techniques, and IP addresses — all logged for analysis.

3
Blue Team
Threat Detected & Logged

Blue Team Python scripts (powered by Scapy) analyze incoming traffic in real time. Abnormal TCP/IP flag patterns trigger immediate alerts and detailed logging of the intrusion attempt.

4
IoT — ESP32
Hardware Alert Fires

Once a threat is confirmed, the ESP32 device triggers a real-world alert — buzzer, LED, or push notification — giving the system administrator instant physical awareness of the attack.

5
Blue Team
Attacker IP Blocked

The Blue Team system automatically blacklists the attacker's IP address, shutting down further intrusion attempts and keeping the protected server safe.
